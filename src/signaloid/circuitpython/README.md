### Helpers for CircuitPython
- `extended_ulab_numpy.py`: Extends the [ulab numpy implementation](https://docs.circuitpython.org/en/latest/shared-bindings/ulab/numpy/index.html) with extra methods needed for the `distributional` and `distributional_information_plotting` modules.
- `plot_wrapper.py`: Implements a plot render specifically for Signaloid Ux histograms.
